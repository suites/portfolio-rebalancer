# ADR 0003: 첫 실거래 범위를 한국 주식 정규 연속매매로 제한

- 상태: 채택
- 결정일: 2026-07-16

## 배경

토스증권 Open API는 한국과 미국 주식 조회·주문을 모두 제공하지만 두 시장은 수량,
통화, 장 운영 시간과 주문 규칙이 다릅니다. 미국 시장은 USD 평가와 환전 비용, 소수
수량, 금액 주문, 현지 거래일과 KST를 넘나드는 세션, 세금·수수료 검증이 추가로
필요합니다. 한국 시장도 KRX와 NXT의 통합 캘린더, 단일가 구간, 호가 단위와 상·하한가를
안전하게 해석해야 합니다.

초기 실거래에서 두 시장을 동시에 허용하면 장애 복구와 주문 대사의 상태 공간이
불필요하게 커집니다. 실제 자금을 다루는 첫 버전은 가장 좁고 검증 가능한 경계를
선택해야 합니다.

## 결정

첫 live 승격 범위는 다음 조건을 모두 만족하는 한국 주식 주문으로 제한합니다.

- 정규 종목 키가 `KR + 6자리 symbol`인 종목
- 토스 종목 조회에서 `ACTIVE`이고 최신 유의사항 재검증을 통과한 종목
- 레버리지·인버스·옵션·공매도·조건주문이 아닌 승인 목록의 현물 종목
- 양의 정수 수량
- `DAY` 유효조건의 지정가 주문
- 토스 KR 통합 캘린더의 `REGULAR_MARKET` 안이면서 단일가 구간 밖
- 최신 가격·호가·상하한가와 별도 검증한 호가 단위에 맞는 가격
- 주문별·일별 극소액 한도, 수동 승인, 킬 스위치와 원장 멱등성을 모두 통과한 주문

토스 캘린더가 KRX와 NXT 정규장의 합집합을 제공하므로 애플리케이션은 특정 거래소를
임의 추론하지 않습니다. 첫 live는 토스가 해당 한국 종목에 적용하는 거래 경로를
사용하되, 프리마켓·애프터마켓과 단일가 구간에서는 신규 주문을 제출하지 않습니다.
캘린더의 단일가 경계가 누락되거나 현재 구간을 확정할 수 없으면 거래를 차단합니다.

다음 기능은 별도 ADR과 승격 검증 전까지 live에서 지원하지 않습니다.

- 미국 주식, USD 주문과 환전
- 소수 수량 또는 금액 기반 주문
- 시장가, 장 마감 주문과 조건주문
- 프리마켓·애프터마켓·단일가 주문
- 고액 주문 확인 플래그가 필요한 주문
- 자동 종목 교체, 레버리지, 인버스, 옵션과 공매도

Shadow와 paper는 미국 보유자산을 평가·표시할 수 있지만, live 계획에 미국 종목이
포함되면 `UNSUPPORTED_LIVE_MARKET`으로 전체 실행을 차단합니다.

## 이유

- 원화와 정수 수량만 다루므로 금액·수량 반올림과 대사가 단순합니다.
- 정규 연속매매와 지정가만 허용하면 체결 가정과 주문 직전 위험 검사가 명확합니다.
- 미국 시장의 소수 수량, 환전, 세금과 현지 거래일 검증을 별도 승격 조건으로 격리합니다.
- 지원하지 않는 세션을 오류로 드러내어 캘린더 결손을 임의 기본값으로 숨기지 않습니다.

## 결과

live executor와 Risk Gate는 시장·세션·수량·주문 유형을 독립적인 필수 조건으로
검사해야 합니다. 옵션 하나만으로 이 제한을 우회할 수 없으며, 미국 시장 지원은
새로운 정책 버전, paper/shadow 비교, 세금·환전 확인과 별도 소액 실거래 검증을
통과해야 합니다.

사용자는 계속 한국·미국 자산을 하나의 장기 목표 포트폴리오로 조회할 수 있지만,
지원하지 않는 live 후보가 있으면 안전하게 계획 또는 실행이 차단되고 한국어 다음
행동이 표시됩니다.
