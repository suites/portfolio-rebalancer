# 로보굴링 공개 가입자 수 정리

동일한 서비스인 로보굴링의 누적 가입자 수로 명시된 공개 시점만 정리했다.

| 기준 시점     | 누적 가입자 | 출처                   |
| ------------- | ----------: | ---------------------- |
| 2023년 2월 말 |    11,000명 | 이데일리, 2023-03-21   |
| 2024년 3월    |    30,000명 | 머니투데이, 2024-03-12 |
| 2024년 5월 말 |    38,000명 | 머니투데이, 2024-06-26 |

2025년 1월 말 119,000명은 로보굴링·연금굴링·주식굴링 합산 수치이므로 같은 시계열에서 제외했다. 각 수치는 삼성증권 발표를 인용한 언론 보도이며 독립적으로 검증된 활성 이용자 수가 아니다.
