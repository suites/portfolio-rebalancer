# Scope

- Audience: 프로젝트 사용자와 구현·검토에 참여하는 개발자
- Goal: 공개 프로젝트명을 `Steady Allocator`에서 `Portfolio Rebalancer`로 일관되게 변경
- Deliverables: `README.md`, `docs/SPEC.md`, `docs/WEB_UI.md` 및 관련 하네스 기록
- Source material: 저장소 내 기존 프로젝트명 사용처
- Freshness: 현재 저장소의 전체 텍스트 검색 결과 기준
- Explicit exclusion: `steady` CLI 명령과 저장소 디렉터리 이름 변경
