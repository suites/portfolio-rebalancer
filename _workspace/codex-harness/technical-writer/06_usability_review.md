# Beginner Usability Review

현재 문서는 개발자 안전 명세로는 강하지만 사용자 안내서로는 부족하다.

필요한 사용자 문서와 기능:

- 자격증명 없이 실행하는 demo Quick Start
- 완전한 `config.example.yaml`
- 토스 키·허용 IP·read-only 연결 점검 절차
- 상태별 의미와 사용자가 해야 할 행동 표
- 종목·비중 변경 전 미리보기
- DB를 직접 수정하지 않는 복구 가이드

사용성 완료 기준은 빈 환경에서 15분 이내 demo 실행, 오류 메시지만 보고 설정 수정, 모든 실행의 한국어 결론 출력, DB 직접 수정 없는 상태 조회다.
